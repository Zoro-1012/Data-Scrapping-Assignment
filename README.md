# AmbitionBox Data Scraping Assignment

## Objective
Scrape structured company data from AmbitionBox for 50 companies and store it in a CSV file.

## How to Run
1. Install dependencies:
   pip install requests beautifulsoup4 pandas

2. Run the script:
   python scraper.py

   This will generate two files:
   - companies.csv (core assignment output)
   - companies_enriched.csv (bonus analysis)
